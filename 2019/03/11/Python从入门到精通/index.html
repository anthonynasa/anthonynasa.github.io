每条if语句的核心都是一个值为True或False的表达式,这种表达式被称为条件测试.
Python并不要求if-elif结构后面必须有else代码块.在有些情况下,else代码块很有用；而
在其他一些情况下,使用一条elif语句来处理特定的情形更清晰

使用break 退出循环

要立即退出while循环，不再运行循环中余下的代码，也不管条件测试的结果如何，可使用

**注意 在任何Python循环中都可使用break语句。例如，可使用break语句来退出遍历列表或字典的for循环。

在循环中使用continue

7.函数

定义函数

下面是一个打印问候语的简单函数，名为greet_user()：

def make_album(name, album, nums=’’):
music = {‘name’: name, ‘album’: album}
if nums:
music[‘nums’] = nums
return music

active = True
while active:
name1 = input(‘输入歌手名字: ‘)
album1 = input(‘输入专辑名名字: ‘)
num1 = int(input(‘输入歌曲数量: ‘))
acd = make_album(name1, album1, num1)
for i, j in acd.items():
print(i, ‘:’, j)
while True:
response = input(‘是否继续(y/n):’)
if response == ‘y’:
break
elif response == ‘n’:
active = False
break
else:
print(‘输入错误,重新输入!’)
continue
